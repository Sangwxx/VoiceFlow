# VoiceFlow 产品定位

VoiceFlow 是纯语音控制的绘图工具。用户可以在专业图表与自由画图模式之间切换，通过语音创建、编辑、管理和导出结构图或 SVG 矢量插画。

## 核心原则

- 语音是主要交互入口，画布保持只读。
- 确定性命令优先走规则路径；复杂意图由 Agent 处理。
- 图表生成与修改在本地校验通过后直接应用；普通缺省信息主动补全，仅在关键歧义会明显改变结果时使用可见反问卡片，并支持语音或文字继续回答。
- 图表数据、视图状态、候选预览和版本数据分别管理，避免状态职责混合。
- 完整新图优先由 LLM 根据受约束的紧凑结构协议生成；模型不负责坐标、样式或内部元数据，本地规划器只在未配置 AI 时降级使用。
- 专业图表与自由画图分别维护作品状态；自由画图第一版使用可编辑 SVG 图元绘制花朵、杯子等简洁插画。

## 当前边界

- 面向本地演示，真实 AI Provider 的密钥会暴露在浏览器中，不适用于生产环境。
- 暂不包含后端同步、多人协作、云端语音识别和更多格式导出。
