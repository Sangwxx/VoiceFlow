# 通用结构图生成

## 目标

- 将完整图表生成从流程图/架构图特例升级为通用结构图生成。
- 支持流程图、架构图、组织结构图、用例图、思维导图、数据流图、框架图和表格等节点关系结构。
- 模型仅输出紧凑结构蓝图，本地负责 ID、校验、样式、布局和 Proposal。
- 确认或取消候选图后，语音任务队列必须继续执行。

## 技术决策

- 继续使用 React Flow 渲染和 ELK 布局，不引入第二套绘图库。
- 所有可表达为节点、分组与连线的结构图统一进入 `create_diagram`。
- 模型输出不包含位置、样式、元数据和完整内部 Diagram 字段。
