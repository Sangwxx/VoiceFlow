# 导出、Agent 容错与上下文边界

## 目标

- 所有导出格式均可通过语言或文字指令在浏览器中实际下载。
- 高频非常规编辑优先走本地 Simple Path，Agent 能容忍带说明文字的 JSON 回复。
- Agent 对话与历史命令严格绑定当前画布，切换画布后不续接旧上下文。

- [x] 审计导出、命令路由、Agent 解析与上下文状态。
- [x] 修复浏览器下载链路并补充导出测试。
- [x] 补充“新增节点”“节点指向节点”等本地编辑语法。
- [x] 增强 Agent JSON 提取与错误恢复。
- [x] 为 Agent 会话和命令历史增加画布边界。
- [x] 完成浏览器验证与全量质量检查。

## 关键决策

- 确定性的导出和基础编辑继续优先使用本地工具，避免不必要的模型延迟与不稳定性。
- Agent 输出解析允许 JSON 外存在少量说明文字，但提取后仍必须通过既有运行时校验。
- 对话上下文以 Diagram ID 为边界；当前画布变化后，旧反问与旧命令历史不可继续使用。
