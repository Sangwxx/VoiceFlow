# 绘图生命周期与大型画布适配

## 目标

- 基础线条和编号图形文字编辑必须走 Simple Path。
- 区分新建基础图形作品与向当前画布追加图形。
- 替换整张画布前自动备份，并提供显式保存按钮。
- 大型组织架构图完成布局后自动完整适配画布。
- 收紧 Agent Operation 协议，避免节点类型被误作操作类型。

## 完成项

- [x] 支持“生成一条线”和“把 5 号圆形上的文字改成学校”。
- [x] 支持一句话创建多个图形并连接。
- [x] 新建图形作品前自动备份，追加类措辞保留当前画布。
- [x] 增加“保存当前图”按钮与自动备份版本类型。
- [x] 大型图表异步布局后重新执行完整视图适配。
- [x] 通过完整质量检查。
