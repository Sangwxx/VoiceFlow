# 自由画图模式

## 目标

- 在画布标题栏提供“专业图表 / 自由画图”按钮切换。
- 两种模式分别保存作品状态，切换时互不覆盖。
- 自由画图模式第一版支持通过语音或文字绘制花朵和杯子。
- 自由画布使用可编辑、可导出的 SVG 图元，而不是生成不可修改的位图。

## 第一版边界

- 支持花朵、杯子以及清空自由画布。
- 支持多个物体依次添加并自动错开放置。
- 暂不实现任意 SVG Path 的 LLM 生成与部件级二次编辑。
