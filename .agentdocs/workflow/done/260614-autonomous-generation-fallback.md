# 完整创建指令主动补全

## 背景

用户说出“画一个流程图”“生成一个最简单的流程图”等完整创建意图时，系统不应因缺少主题、节点或步骤细节而停止执行。当前真实 AI 可能返回 `clarification`，执行器随后统一显示“指令信息不足”，与产品“不使用系统反问阻塞”的原则冲突。

## 目标

- 创建图表请求只要明确表达创建意图与图表类型，就直接执行。
- AI 未主动补全时，使用本地结构图规划器可靠降级。
- “最简单的流程图”生成真正精简的默认流程。
- 新的有效识别结果到达后，清除已过时的语音网络错误，避免遮盖命令执行结果。

- [x] 完善本地最简流程图规划
- [x] 清理过时语音识别错误并补充测试
- [x] 完成全量质量检查并推送
