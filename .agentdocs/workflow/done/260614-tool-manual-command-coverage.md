# 工具手册命令逐条覆盖

## 背景

工具手册展示了画布控制、图形编辑、结构图生成、美化、版本管理和导出命令，但示例与路由、执行器测试没有建立强绑定，可能出现手册写了但实际不可执行的情况。

## 目标

- 将工具手册示例作为可复用的单一数据源。
- 为每条示例建立明确的预期路由和执行覆盖。
- 确定性工具操作优先使用本地 Fast / Simple / Workflow Path。
- 结构图生成继续由 LLM 主导，并提供未配置 AI 的本地降级。
- 补齐失败命令并确保手册不再展示无法执行的示例。
