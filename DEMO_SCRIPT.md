# VoiceFlow 三分钟参赛 Demo 脚本

## 录制前准备

1. 在 `.env.local` 配置真实 `VITE_AI_BASE_URL`、`VITE_AI_API_KEY` 和 `VITE_AI_MODEL`，重启开发服务器。
2. 使用桌面版 Chrome 或 Edge，提前允许麦克风权限。
3. 刷新页面，确认初始图为“用户登录流程”，点击“开始语音输入”并允许麦克风权限。
4. 全程不使用鼠标、键盘或触摸编辑画布。

## 录制流程

### 0:00–0:25 产品与创新点

旁白：

> VoiceFlow 是一款纯语音 AI 绘图工具。它通过分层路由降低常用命令延迟，通过本地语音校准提高指令识别准确率。

指出右侧“智能决策指标”和画布上的“纯语音只读画布”。

### 0:25–0:50 Fast Path 低延迟直通

1. 说：`横向布局`
2. 说：`撤销`
3. 说：`导出 SVG`

指出右侧执行路径为 `FAST`，并展示本次耗时、平均耗时与 Fast Path 命中率。

### 0:50–1:20 上下文语义纠错

1. 说：`声成一张强化学西的流成图`
2. 指出底部字幕将口误纠正为“生成一张强化学习的流程图”，并显示纠错置信度。
3. 展示校准后的指令直接生成结构图。

### 1:20–1:55 确定性目标匹配

1. 说：`把失败分支改成红色虚线`
2. 指出系统按本地相关性排序选择最佳候选并直接执行，不发起 AI 请求。
3. 展示匹配的失败分支变为红色虚线。

### 1:55–2:35 通用结构图直接生成

1. 说：`画一个学生选课用例图`
2. 指出 LLM 根据主题规划结构蓝图，本地校验并自动排版后直接应用。
3. 说：`保存当前版本叫演示完成`

### 2:35–3:00 总结

旁白：

> VoiceFlow 让确定性命令快速执行，让语音校准在本地完成，让复杂 AI 修改经过校验后直接应用，在纯语音场景中兼顾速度、容错和连续操作。

最后展示智能决策指标、版本记录和完整画布。
录制结束后点击“停止语音输入”关闭麦克风监听。

## 录制失败备用

- 麦克风识别不稳定：重新录制该段，本地校准会优先修正常见错词。
- 浏览器拒绝麦克风：在地址栏权限设置中允许后刷新。
- Agent 请求失败：检查 `.env.local` 的真实 Provider 配置和网络状态。
