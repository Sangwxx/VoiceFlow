# VoiceFlow Pure Voice Demo Script

This script requires no mouse or keyboard interaction after microphone permission is granted.

1. Say: `画一个用户登录流程图`
2. Say: `确认`
3. Say: `在登录页后面加一个验证码校验节点`
4. Say: `把失败分支改成红色虚线`
5. Say: `聚焦验证码校验`
6. Say: `看全图`
7. Say: `撤销`
8. Say: `重做`
9. Say: `保存当前版本叫修改完成`
10. Say: `整理成适合汇报的版本`
11. Say: `确认`
12. Say: `隐藏异常分支`
13. Say: `显示异常分支`
14. Say: `对比版本修改完成`
15. Say: `导出 SVG`
16. Say: `导出 PNG`

Alternative scene command: `加载电商订单演示场景`, then say `确认`.
